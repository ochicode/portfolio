# Software Engineer Portfolio

A modern, responsive portfolio website designed for software engineers and tech professionals.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Comprehensive Sections**:
  - About Me: Professional introduction
  - Technical Skills: Organized by technology category
  - Featured Projects: Showcase of key projects with tech stacks
  - Experience: Professional timeline
  - Contact: Links to social profiles and email

## Getting Started

Simply open `index.html` in your web browser to view the portfolio.

### Viewing Locally

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

### Deploying

This is a static website that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Customization

Edit the following files to personalize your portfolio:

- `index.html`: Update content, add your information
- `styles.css`: Modify colors, fonts, and styling
- `script.js`: Add custom JavaScript functionality

### Quick Customization Guide

1. **Replace placeholder content** in `index.html`:
   - Update the hero section with your title and tagline
   - Modify the About section with your personal story
   - Add your actual skills in the Skills section
   - Replace project examples with your real projects
   - Update experience with your work history
   - Change contact links to your actual profiles

2. **Customize colors** in `styles.css`:
   - Update the gradient colors in `.hero` section
   - Modify accent colors (currently purple/blue theme)

3. **Add your personal branding**:
   - Update the nav brand name
   - Add your logo or profile picture

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Responsive Web Design
